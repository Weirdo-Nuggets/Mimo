  • The Box Model
    1. Tier 1- Content
      • To change the alignmentof the content, we use the "text-align" property
    2. Tier 2- Paddings- They generate space between the content and the border of an element
      • To change the paddings on all four sides of an element, we use the "padding" property
      • To change the padding on the left side of an element we use the "padding-left" property
      • To change the padding on the right side of an element we use the "padding-right" property
      • To change the padding at the top side of an element we use the "padding-top" property
      • To change the padding at the bottom side of an element we use the "padding-buttom" property
    3. Tier 3- Borders-
      • To change the alignmentof the content, we use the "text-align" property
    4. Tier 4- Margins- They generate space around an element, outside of any paddings and borders
      • To change the margins on all four side of an element, we use the "margin" property
      • To change the margin at the top of an element, we use the "margin-top" property
      • To change the margin at the bottom of an element, we use the "margin-bottom" property
      • To change the margin on the left side of an element, we use the "margin-left" property
  • Difference
    1. Margins vs. Paddings- The padding adds space between the content and the border, while the margin adds space between the border and other elements
