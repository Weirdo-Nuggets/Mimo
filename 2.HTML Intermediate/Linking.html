Notes


  • Terms
    1. Index.html- the home page of the website
    2. Webpage- A file woth .html at the end
    3. Website- A group of webpages in one folder all linked to each other
    4. Internal Linking- Linking the webpage to another webpage on your domain/folder/directory
    5. External Linking- Linking the webpage to another webpage on the WWW
  • Notes
    1. To make things easier to read, we lowercase each letter and put a hyphen between each space
    2. target values-
      1. _parent -Opens file on the same webpage, in a container (opens in the same webpage)
      2. _blank - Opens file on another tab
      3. _top -Opens the file in the full body of the same webpage (opens in the same webpage)
      4. _self -Opens in the same webpage its the default value

All of these are internally linked, because they are all in one folder
