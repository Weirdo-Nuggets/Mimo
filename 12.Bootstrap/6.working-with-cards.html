NOTES

  • Meaning
  • To create a Bootstrap card, code the class="card" into a div(it needs to be a div!)
  • To have an image on top of the card use card-img-top
  • To create a card body section that will contain heading and text use card-body inside a separate div
  • The card-text class defines text in cards
  • Use card-header to have a header section in the card
  • Use the card-footer class to add footer section to the card
