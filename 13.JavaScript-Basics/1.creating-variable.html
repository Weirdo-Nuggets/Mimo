NOTES

  • Javascript is a scripting language that runs in the browser.
  • JS is behind every action/interaction you have performed on the browser
  • Variables are basically boxes that have content and names in them that tell us what they mean

  • Variables are written as var
  • Each variable's name need to be spaceless, so you can write it in camel case (home-city or homeCity)
  • Try giving the names, descriptive names so you know what's inside
  • If there are multiple variables you can add names

  • To finish off the variable, add ="value";
  • Values in double quotes are called strings ("Chicago")

  • The instruction console.log(display);will tell the computer to display a value in an area called the console
  • If you are referring to a name not linked to a variable use "", if you're referring to a variable don't use quotes
  • So it is important that the lines of instruction are structured in the way they should run
