NOTES

  • To return a value from a function use the return keyword followed by the value to be returned
  • It is similar to calling it directly
  • The returning process is tricky.
    1. Name the function and give it a variable to refer to
    2. Then create a variable that will trigger the function and then give the function a value that refers to the initial variable
    3. Then console the trigger variable, which will return the initial variable
  • If you don't put the return, running it will not work - will show as undefined
