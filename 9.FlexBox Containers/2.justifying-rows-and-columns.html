Notes

  • Rows
    • To flex items at teh center of the main axis use justify-content:center;
    • the evenly space out the content (margins) use justify-content:space-around;
    • To keep space between items use justify-content:space-between;
  • Columns
    • The same values except they can only be visible if the container is taller than all the items
