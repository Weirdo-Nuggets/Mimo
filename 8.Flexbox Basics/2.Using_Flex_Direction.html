Notes

  • Meanings- Flex Direction
    • To control the direction elements are displayed, the flex-direction property is sued.
    • flex-direction: row/column. Row is teh default
    • the row value means items start from the left of the container, horizontally(y'know the default
      but the content stretches out to fill the container's height, so really confusing
    • the column value means items start from the top of the container, vertically (y'know like a column).
      Only the items stretch out to fill the container's width which makes it not look like a column...so confusing -_-
    • In conclusion, flex-direction:row looks like columns and flex-direction: column looks like rows so have fun!
  • Meanings- Reverse Direction
    • To make the items start at the right side of container use the flex-direction:row-reverse; declaration
    • To reverse the starting position of columns, use flex-direction:column-reveerse;
