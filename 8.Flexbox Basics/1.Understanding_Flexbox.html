Notes

  • Meanings
    • With Flexbox, elements adapt in size to fit perfectly
      on different scenes.
    • To apply it, there needs to be a parent element (flex container) with a nested child element (flex item) inside
    • property:value = display:flex
  • Behaviour
    • By default, Flexbox fits items inside a container on 1 row,
      and flex items automaticallt stretch to fill their container's height.
      Flex items are as wide as the content inside of them, and can shrink to
      fit all other items in one container
    • If content that pass the total width of the container are coded, that item will overflow
    • display:flex affects inline and block elements alike, it ain't racist :)
    • In conclusion they will grow taller but will refuse to grow bigger
