Notes

  • Meaning
    • Use the grid-template-row: value; property to adjust the size of grid rows
    • Unlike columns if there are fewer items than rows we will get empty rows
    • Percentages can also be used but only if the container has a fixed size
    • Like with columns, we can use px and fr at the same time
    • Setting the row size to auto will make that row take up as much space as there is available
    • Like columns we can use the repeat function. where;
      repeat(3, 40px) this means that 3 rows take up 40px each
