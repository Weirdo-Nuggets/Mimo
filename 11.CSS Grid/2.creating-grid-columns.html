Notes

  • Meaning
    • To adjust the size of columns on a CSS Grid use the normal values
    • Grid items are displayed based on the no. of columns.
      If there are more items than columns, extra items will be pushed down to the next row.
    • Setting a column to auto will make grid items take up as much space as they can
    • Fractional unit or fr distributes the container's avaliable space in fractions
    • If we combine fr with px e.g, the space left after adding the px item is what gets distributed in fractions.
    • Using the repeat(3, 1fr) function means that 3 columns will take up 1fr each
    • Using px fixes the size of columns so it is not good for responsive webpages
