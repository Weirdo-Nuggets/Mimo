Notes

  • Meaning
    • The CSS Grid is a layout system that lets us add items both horizontally and vertically on a webpage.
    • Unlike Flexbox where we can only place items on one dimension at a time
    • Like Flexbox it needs a parent container
    • Like Flexbox we need the display: grid; declaration
    • To organize grid items into columns use grid-template-columns: value; grid-template-columns: 50px 50px 120px;
    • The value is based on each column's size so;
