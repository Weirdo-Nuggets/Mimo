Notes

  • Meaning
    • They are used to label certain part of the webpage such as the header, footer etc
    • Use the grid-area: classname; declaration to assign an area name
    • Firstly, different areas should have different class names. This is shows in the
      grid-template-areas: classname classname classname; declaration which assigns an item to the grid layout
    • To span a grid item across multiple grid columns, repeat the name of the grid item
    • Using . or ...leaves a grid area empty
    • To adjust the order in which it is shown use "areaname". All of the area spaces must be filled
      otherwise it will distort the grid
    • The first line of names represents 1 column, and the first item represents one row
    • Code the rows on different lines so it is easier to read
