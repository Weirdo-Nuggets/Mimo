  • Loops count upwards...up to down, but what if they counter downwards
  • Basically for ( var i = 0; i < 4; i++) { console.log(i); }
  • But backwards for ( var i = 5; i > 0; i--) { console.log(i); }
  • You want to include 0 when counting downwards...simple use >=
