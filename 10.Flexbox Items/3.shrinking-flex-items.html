Notes

  • Meaning
    • Flex items shrink when they can't expand to their flex-basis value using flex-shrink: 1; as the default
    • Setting the value to 0 means the item does not shrink.
    • If both items are set to 0, they overflow outside the container's borders
    • Just like flex-grow, flex-shrink can be set to overpower other items but the items need to be bigger in other to shrink more
