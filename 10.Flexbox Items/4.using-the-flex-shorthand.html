Notes

  • Meaning
    • The default values for the flex item properties are
      • flex-grow: 0;
      • flex-shrink: 1;
      • flex-basis: auto;
    • So we can just use the flex shorthand to write all of them like margins, paddings and border-radius
      • flex: 0 1 auto;
