NOTES

  • Booleans are conditions that make decisions on whether to run or skip lines of code. They come in (true/false/variable)
  • Statements relying on conditions are called conditional statements
  • A code that adapts to different situations is the if statement
  • By using (true) we're are telling the browser to run the code only if (string or number) is true or exists.
  • If (false) then it should not run the variable if it exists. Thus, skipping the code.

  • If statements decide/don't to run the entire code, and only make decisions on a piece of code. This piece is called a code block.
  • This means that if we do not add the {} in between the code, it makes the condition run/skip the entire code
  • If we put the {} around the code like console.log(), we run only that instance of code, which is called a code block
  • The code block can be +1 lines
  • You know the drill, we can store the value of a condition in a variable
