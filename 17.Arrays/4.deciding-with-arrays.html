NOTES

  • the word length will count the no. of elements in the array, even if there are non
  • To display this write the varName.length inside the console
  • Like always this length can be stored in a variable
  • The length of an array can be used to create conditional statements
  • You can also use for loops to show numerically how many elements there are, as they will repeat
  • Instead of specifying that i < 3 we can use the friends.length in less so i < friends.length;
  • Here's a fun realization, since i also means index, guess what else means index...that's right the array index
  • So each time the i repeats, it actually repeats the index values in the arrays, so we can use this to actually repeat and show the actual values instead of numbers
